# 自然码双拼键位图

自然码双拼码表渲染，单页展示，移动端自动适配横屏查看。

在线预览：[ziranma.inshomepgup.com](https://ziranma.inshomepgup.com)

## 技术栈

- Vue 3 + TypeScript
- Vite
- UnoCSS

## 快速开始

```bash
pnpm install
pnpm dev
```

## 其他命令

```bash
pnpm build     # 类型检查 + 构建
pnpm preview   # 预览构建产物
pnpm lint      # ESLint 检查
pnpm lint:fix  # ESLint 自动修复
```

## 部署

推送到 `main` 分支后，GitHub Actions 会自动构建并部署到 GitHub Pages。
